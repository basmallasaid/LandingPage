##  Installation  

1. Clone the repo:  
   ```bash
  https://github.com/basmallasaid/LandingPage
   cd velora
````

2. Install dependencies:

   ```bash
   npm install
   ```

3. Run development server:

   ```bash
   npm run dev
   ```

4. Build for production:

   ```bash
   npm run build
   ```

##  Customization

* Replace images in the `public/` folder with your brand assets.
